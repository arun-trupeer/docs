This document provides a step-by-step guide to creating a shareable page in Trupeer. It covers selecting content, customizing sharing options, and generating an embed code for seamless integration with social media platforms or websites.

### Step 1: Access Sharing Options

Click on the **Share** option.

![Screenshot 1](images/create-page/image1.jpg)

### Step 2: Choose Page Sharing Feature

And select **share as page feature**.

![Screenshot 2](images/create-page/image2.jpg)

### Step 3: Set Page Title

Then enter a **title that suits** your requirements,

![Screenshot 3](images/create-page/image3.jpg)

### Step 4: Select Content

Then, select the video and document content you wish to share.

![Screenshot 4](images/create-page/image4.jpg)

### Step 5: Save Your Changes

Click on **save Changes**.

![Screenshot 5](images/create-page/image5.jpg)

### Step 6: Open Social Sharing Menu

Then click **on social option**,

![Screenshot 6](images/create-page/image6.jpg)

### Step 7: Choose Sharing Platform

Then choose a **platform to share** the document.

![Screenshot 7](images/create-page/image7.jpg)

### Step 8: Select Embed Option

Click on **Embed**,

![Screenshot 8](images/create-page/image8.jpg)

### Step 9: Copy Embed Code

And **click on copy**.

![Screenshot 9](images/create-page/image9.jpg)
